# CREOVA — AI Creator Operating System

Full-stack AI SaaS for content creators. Next.js 15 + Supabase + Razorpay.

## Stack
- Next.js 15 App Router + TypeScript
- Tailwind CSS + custom design system  
- Supabase (Auth + DB + Storage)
- Razorpay payments
- Claude AI (content generation)

## Quick Start
```bash
npm install
cp .env.local.example .env.local
# Fill in keys
npm run dev
```
